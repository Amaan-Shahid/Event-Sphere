# EventSphere

University event, registration, volunteer, attendance, and certificate management system.

## Tech Stack

- Backend: Node.js, Express, MySQL
- Frontend: React, Vite, Vanilla CSS
- Other: JWT, Puppeteer (for certificates), Multer (file uploads)

## Structure

- `/server` — backend
- `/client` — frontend (to be added)
- `/sql` — database schema & seeds
